## q8 開発許可 (`kaihatsu-kyoka.ts` / 都市計画法29条・37条) 🚩

**根拠** — 市街化区域は原則1,000㎡以上で許可(未満は不要)。市街化調整区域は面積の下限なし・原則許可必要。図書館・公民館・変電所等の公益建築物は規模問わず許可不要。工事完了公告前は原則建築禁止(37条)。

| 肢 | 判定 | 照合 | 🚩数字 |

| 1 市街化区域は原則1,000㎡以上で許可=○ | 1,000㎡以上 | ✅ | 1,000㎡ |

## q9 建蔽率・容積率 (`kenpei-yoseki.ts` / 建築基準法53条・52条)

**根拠** — 建蔽率80%の地域かつ防火地域内の耐火建築物は制限なし(53条)。前面道路12m未満は指定容積率と「幅員×法定乗数」の小さい方(52条)。敷地が2用途地域にわたる建蔽率・容積率は面積按分(加重平均)。角地+10%緩和は建蔽率(容積率にはなし)。

| 肢 | 判定 | 照合 |
|---|---|---|
| 1 80%地域かつ防火地域耐火建築物→制限なし=○ | 53条 | ✅ |
| 2 前面道路12m未満→小さい方が限度=○ | 52条 | ✅ |
| 3 2地域にわたる容積率は過半主義=×(誤り) | 建蔽率・容積率は面積按分 | ✅ |
| 4 角地は容積率+10%=×(誤り) | 角地緩和は建蔽率 | ✅ |

**総合: 妥当。** 「按分は建蔽率容積率/過半主義は用途制限」の対比が正確。承認: ☑ verified(Mami 2026-07-16) / ☐ 要修正

## q10 容積率(延べ面積) (`yoseki-calc.ts` / 建築基準法52条・calc) 🚩

**根拠** — 前面道路12m未満は指定容積率と「幅員×法定乗数(住居系4/10)」の小さい方。延べ面積上限=敷地面積×適用容積率。

| 段 | 計算 | 検算 |
|---|---|---|
| ① 道路容積率 | 6m×4/10=240% | 🚩 ✅ |
| ② 小さい方 | 240%＜300%→適用240% | ✅ |
| ③ 延べ面積 | 400㎡×240%=960㎡ | 🚩 ✅ |

**総合: 妥当。** distractor(指定300%そのまま・乗数6/10誤用・除算)も的確。答え960㎡は検算OK。承認: ☑ verified(Mami 2026-07-16) / ☐ 要修正
