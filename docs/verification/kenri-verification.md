## q1 二重譲渡 (`nijuu-jouto.ts` / 民法177条)

**根拠原文** — 民法177条:「不動産に関する物権の得喪及び変更は、不動産登記法…その他の登記に関する法律の定めるところに従いその登記をしなければ、第三者に対抗することができない。」

| 肢 | 問題の判定 | 照合 |
|---|---|---|
| 1 先に登記したCがBに対抗できる=○ | 登記の先後で決まる | ✅ |
| 2 先に契約したBは登記なしでも対抗できる=×(誤り) | 対抗には登記が必要 | ✅ |
| 3 単なる悪意のCは登記しても勝てない=×(誤り) | 単なる悪意者は保護、負けるのは背信的悪意者だけ | ⚠️判例 |
| 4 背信的悪意者は登記しても勝てない=○ | 背信的悪意者は177条の「第三者」にあたらない | ⚠️判例(最判昭43.8.2 ほか) |

**総合判定: 妥当。** 条文＋確立した判例法理と一致。

## q2 詐欺・強迫と第三者 (`sagi-kyouhaku.ts` / 民法96条)

**根拠原文** — 民法96条3項(2017改正後):「前二項の規定による詐欺による意思表示の取消しは、**善意でかつ過失がない**第三者に対抗することができない。」(強迫には第三者保護規定なし)

| 肢 | 問題の判定 | 照合 |
|---|---|---|
| 1 詐欺で取消しできる=○ | 96条1項 | ✅ |
| 2 善意無過失のCに取消しを対抗できない=○ | 96条3項の文言どおり(2017改正で「無過失」明記) | ✅ |
| 3 強迫でも善意無過失Cに対抗できない=×(誤り) | 強迫に第三者保護規定なし→対抗できる | ✅ |
| 4 悪意のCでも登記あれば対抗できない=×(誤り) | 取消し前は善意無過失が要件。悪意者は登記あっても不保護 | ✅ |

**総合判定: 妥当。** 2017改正の「善意でかつ過失がない」も正しく反映。

| 肢 | 問題の判定 | 照合 |

## q18 意思表示(虚偽表示・錯誤・心裡留保) (`ishi-hyouji.ts` / 民法93・94・95条)

**根拠原文** — 94条:通謀虚偽表示は無効、2項で善意の第三者に対抗不可 / 95条3項:表意者に重過失があると原則取消し不可(相手方悪意・重過失、共通錯誤は例外) / 93条:心裡留保は原則有効。

| 肢 | 問題の判定 | 照合 |
|---|---|---|
| 1 通謀の仮装売買はAB間で無効=○ | 94条1項 | ✅ |
| 2 善意なら過失があってもAは無効を対抗できない=○ | 94条2項。第三者は善意で足り無過失・登記不要 | ⚠️判例・通説(無過失必要説もあるが宅建標準は善意のみ) |
| 3 重過失でも錯誤取消しできる=×(誤り) | 95条3項で原則取消し不可 | ✅ |

## q20 借地借家法(存続期間・更新・定期借家) (`shakuchi-shakka.ts` / 借地借家3・4・29・38条)

**根拠原文** — 3条:借地権の存続期間は30年(契約でより長くは可)/ 4条:更新後は初回20年・2回目以降10年 / 29条1項:1年未満の建物賃貸借は「期間の定めがない」とみなす / 38条:定期借家は書面(電磁的可)で契約、事前に更新なしを記載した**別個の書面**を交付して説明、欠くと更新なしの定めが無効。

| 肢 | 問題の判定 | 照合 | 🚩数字 |
|---|---|---|---|
| 1 借地25年の定めは有効=×(誤り、30年になる) | 3条=30年、短い定めは30年に引上げ | ✅ | 25→30年 |
| 2 最初の更新で定めなし→20年=○ | 4条=初回更新20年 | ✅ | 20年 |
| 3 普通借家10か月は有効=×(誤り、期間の定めなし) | 29条1項=1年未満は期間の定めなしとみなす | ✅ | <1年 |
| 4 定期借家で事前書面説明を欠く→更新なしの定めが無効=○ | 38条2項(旧3項)=事前説明書面。説明欠くと更新なし定めが無効、契約は普通借家に | ✅ | — |

**総合判定: 妥当。** 数字(30/20/10年、1年)すべて条文と一致。肢4「契約書とは別個の書面が必要=判例」も裏取り済み。
