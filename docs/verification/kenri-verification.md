## q59 不法行為(使用者責任・工作物責任・消滅時効) (`fuhou-koui.ts` / 民法709条・715条・717条・724条)

**根拠原文**

- 民法715条1項:「ある事業のために他人を使用する者は、被用者がその事業の執行について第三者に加えた損害を賠償する責任を負う。ただし、使用者が被用者の選任及びその事業の監督について相当の注意をしたとき…は、この限りでない。」/ 同3項:「前二項の規定は、使用者又は監督者から被用者に対する求償権の行使を妨げない。」
- 民法717条1項:「土地の工作物の設置又は保存に瑕疵があることによって他人に損害を生じたときは、その工作物の占有者は、被害者に対してその損害を賠償する責任を負う。ただし、占有者が損害の発生を防止するのに必要な注意をしたときは、所有者がその損害を賠償しなければならない。」
- 民法724条:「不法行為による損害賠償の請求権は、次に掲げる場合には、時効によって消滅する。一 被害者又はその法定代理人が損害及び加害者を知った時から三年間行使しないとき。二 不法行為の時から二十年間行使しないとき。」/ 724条の2(人の生命・身体を害する不法行為は前条1号を「五年間」に読み替え)。

| 肢 | 問題の判定 | 照合 |
|---|---|---|
| 1 賠償した被用者Bは使用者A社に相当額を求償できる=○ | 逆求償を肯定 | ⚠️判例(最判令2.2.28) |
| 2 賠償した使用者A社はBに全額を当然に求償できる=×(誤り) | 求償は信義則上相当な限度に制限 | ⚠️判例(最判昭51.7.8) |
| 3 占有者かつ所有者のA社は必要な注意の証明で免責される=×(誤り) | 占有者免責時は所有者が無過失責任 | ✅(717条1項ただし書) |
| 4 生命身体侵害以外の請求権は知った時から3年で時効消滅=○ | 724条1号どおり | ✅ |

**総合判定: 妥当(AI下書き)。** 使用者責任の逆求償(令和2年判例)・求償制限(昭和51年判例)、工作物責任の占有者免責→所有者無過失責任の構造、724条の3年/20年(724条の2で生命身体は5年)を条文・判例原文と突き合わせ済み。肢3は「占有者かつ所有者」の場合に占有者免責でも所有者として免れない点が狙い。

残る確認推奨ポイント:

- 逆求償(肢1)は令和2年最判の新しい判例法理。承認時に判旨の原文確認を推奨
- 715条3項の求償制限(肢2)は条文にない判例法理(⚠️)

## q60 債権譲渡・相殺(譲渡制限特約・対抗要件) (`saiken-jouto.ts` / 民法466条・467条・509条)

**根拠原文**

- 民法509条:「次に掲げる債務の債務者は、相殺をもって債権者に対抗することができない。…一 悪意による不法行為に基づく損害賠償の債務 二 人の生命又は身体の侵害による損害賠償の債務…」

| 2 譲受人C自身の通知で対抗できる=×(誤り) | 通知の主体は譲渡人(467条1項) | ✅ |

| 4 加害者Bが悪意の不法行為による賠償債務を受働債権に相殺できる=×(誤り) | 509条1号で相殺禁止 | ✅ |

## q61 危険負担・同時履行の抗弁(契約総論) (`kiken-futan.ts` / 民法533条・536条・567条)

**根拠原文**

- 民法536条1項:「当事者双方の責めに帰することができない事由によって債務を履行することができなくなったときは、債権者は、反対給付の履行を拒むことができる。」/ 同2項:「債権者の責めに帰すべき事由によって債務を履行することができなくなったときは、債権者は、反対給付の履行を拒むことができない。この場合において、債務者は、自己の債務を免れたことによって利益を得たときは、これを債権者に償還しなければならない。」
- 民法567条1項:「売主が買主に…目的物…を引き渡した場合において、その引渡しがあった時以後にその目的物が当事者双方の責めに帰することができない事由によって滅失し、又は損傷したときは、買主は、その滅失又は損傷を理由として、履行の追完の請求、代金の減額の請求、損害賠償の請求及び契約の解除をすることができない。この場合において、買主は、代金の支払を拒むことができない。」
- 民法533条:同時履行の抗弁。546条:解除の原状回復義務に533条を準用。

| 肢 | 問題の判定 | 照合 |
|---|---|---|

| 2 引渡し後・双方無責の滅失でも買主は代金支払を拒める=×(誤り) | 567条1項で危険は買主に移転 | ✅ |
| 3 解除による双方の原状回復義務は同時履行=○ | 546条・533条 | ✅ |
| 4 買主の帰責事由で引渡し不能→買主は代金支払を拒める=×(誤り) | 536条2項で拒めない | ✅ |

**総合判定: 妥当(AI下書き)。** 危険負担の履行拒絶権構成(536条1項・2020改正で旧534条債権者主義を廃止)、引渡しによる危険の移転(567条1項)、債権者帰責時の拒絶不可(536条2項)、解除の原状回復と同時履行(546条)を条文原文と突き合わせ済み。肢1↔肢2(引渡しの前後)、肢1↔肢4(双方無責と債権者帰責)で結論が逆転する対比が狙い。
